# React TypeScript PDF Demo

Kısa demo: PDF yükleme arayüzü. Projeyi başlatmak için:

```bash
cd react-ts-demo
npm install
npm run dev
```

Notlar:
- Bu başlangıç demo yalnızca PDF seçmeyi gösterir; PDF içeriğini analiz etme fonksiyonu henüz eklenmedi.
- İsterseniz PDF analizini otomatikleştirip sonuçları gösterecek şekilde genişletebilirim.
